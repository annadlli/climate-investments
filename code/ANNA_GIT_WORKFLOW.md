# Anna Git Workflow

Anna-owned note for automating repo pull, commit, and push requests.

Do not edit this file unless Anna explicitly asks.

## Repository Defaults

- Repo path: `/Users/Anna/Desktop/climate-investments`
- Branch: `main`
- Remote: `origin`
- Remote URL: `https://github.com/annadlli/climate-investments.git`

## Must Do Each Git Session

1. Check repo state before changing Git state:

```bash
git status --short --branch
```

2. For pull/refresh requests, use:

```bash
git fetch origin
git pull --rebase --autostash
```

3. For commit/push requests, show status first, then stage only what Anna requested:

```bash
git status --short
git add <requested files>
git commit -m "<message>"
git push origin main
```

4. Verify push landed:

```bash
git status --short --branch
git log origin/main -1 --oneline
```
